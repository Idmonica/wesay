Welcome to WeSay 1.9.x-Alpha

You have installed an alpha version of WeSay.  New features appear in the alpha versions, but new bugs might appear too! Let us know of any problems.

About this version

WeSay comes in two flavors: 'stable' and 'alpha/beta'.   Once a version is 'stable", it will only receive critical bug fixes, but not new features or anything risky.

If crashes or other bugs would be a major problem for you, please uninstall this version and get our most recent stable one (one without the 'beta' tag).  If you are helping us advance WeSay, are evaluating WeSay's capabilities, or are just willing to risk the occasional bug, this is the version for you.  This is where we are currently adding new abilities in response to your requests, and we need people to help ensure that it can become stable, too. If you report a bug via issues@wesay.org, we can oftentimes fix it and provide a new release within hours.  So let us know if you're in a hurry for a fix.

What's New in 1.9?

Version 2.0 is a synchronized release with FieldWorks which migrates user and project preferences from the writing system LDML (Locale Data Markup Language) file to a new SharedSettings folder. Common writing systems between the two tools will be able to share font and keyboard selections. Note: this migration is not backwards-compatible with prior versions of WeSay. Ensure the rest of the collaborators on your team that send/receive the project also upgrade to the same versions.

It is recommended to be online when creating a new project so WeSay can download the associated LDML from the SLDR (SIL Locale Data Repository).

Sound input systems do not work on some computers.

Ready to go deeper?

Consider joining the WeSay Language Software Community site.   Until we have real documentation, this is the place to go when something is unclear.  As you ask questions, we'll answer them there.  You can read old questions and answers there.

Please report bugs or request a feature using email. Tell us that you tried it and what you did.  No, really.  It's human nature to assume someone else had the same crash, but if it was easy to run into, we would have found it, and we would have fixed it.    When you send to this address, your issue is automatically entered into our "todo list" and you will get updates as we work on the problem.

We are adding to WeSay's abilities regularly, and publishing new versions here.  Major new features are always announced on the WeSay news.
