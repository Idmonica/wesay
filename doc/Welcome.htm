Welcome to WeSay 1.6.x-Alpha

You have installed an alpha version of WeSay.  New features appear in the alpha versions, but new bugs might appear too! Let us know of any problems.

About this version

WeSay comes in two flavors: 'stable' and 'beta'.   Once a version is 'stable", it will only receive critical bug fixes, but not new features or anything risky.

If crashes or other bugs would be a major problem for you, please uninstall this version and get our most recent stable one (one without the 'beta' tag).  If you are helping us advance WeSay, are evaluating WeSay's capabilities, or are just willing to risk the occasional bug, this is the version for you.  This is where we are currently adding new abilities in response to your requests, and we need people to help ensure that it can become stable, too. If you report a bug via issues@wesay.org, we can oftentimes fix it and provide a new release within hours.  So let us know if you're in a hurry for a fix.

What's New in 1.6?

Version 1.6 migrates user and project preferences from the writing system LDML (Locale Data Markup Language) file to a new SharedSettings folder that will be accessible to future versions of other lexicon tools. Once a project has been migrated in WeSay 1.6, the data will be incompatible to prior versions of WeSay.

It is ideal to be online when creating a new project so WeSay can download the associated LDML from the SLDR (SIL Locale Data Repository).

What's New in 1.5?

Version 1.5 mainly brings the functionality of the Windows version to the Linux platform. However, some improvements were made in the Windows version as well.

Common improvements

Improved setup of keyboards in the Configuration Tool.

Now possible to start a new project from Chorus Hub in the Configuration Tool.

Fixed some display issues in Dictionary Browse & Edit.

Many bugfixes for Send/Receive with FLEx.

Fix some issues with project creation.

Improvements to the Windows version

Installer now includes Gecko for rendering of complex scripts. It is off by default, but can be turned on.

New in the Linux version

Fix rendering issues for both roman and non-roman scripts.

Enable use of ibus keyboards.

Fix many display issues (missing icons, clipped labels, etc.).

Fix the selection of the semantic domain.

Enable add-ins.

Export to PDF and LibreOffice.

Enable several features that were new since WeSay 1.1:

  • Audio recording and playback
  • Chorus Notes
  • Art of Reading image collection
  • Send/Receive capability
